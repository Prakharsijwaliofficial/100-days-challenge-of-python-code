📅 Day 01 – Band Name Generator 🎸
🧠 Project Overview
This simple Python script generates a fun band name by combining the name of your hometown with your pet’s name. It’s a classic beginner project that introduces user input, string manipulation, and basic printing.
🛠️ What It Does
- Greets the user.
- Asks for two inputs:
- The city they grew up in
- The name of a pet
- Combines both inputs to suggest a band name.

📄 Code
print("Welcome to the Band Name Generator.")
city = input("Which city did you grow up in?\n")
pet = input("What is the name of a pet?\n")
print("Your band name could be " + city + " " + pet)

🧪 Sample Output
Welcome to the Band Name Generator.
Which city did you grow up in?
Haldwani
What is the name of a pet?
Shadow
Your band name could be Haldwani Shadow


🚀 What I Learned
- How to use input() to get user data.
- How to store and combine strings.
- How to format output for a fun user experience.
